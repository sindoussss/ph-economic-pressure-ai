TITLE: Maria Clara
YEAR: 2026

Maria Clara AI is an offline-capable AI assistant designed to provide
knowledge and emergency-related information.

Maria Clara is created by a group of student from Lyceum of the Philippines University Manila (LPU M).

Maria uses:
- Local document retrieval (RAG)
- Local Wikipedia knowledge
- A language model for response generation
- A local model that can seek emergency if needed

Maria is mostly for offline use but Maria has hybrid mode where she can access the web if the user have internet.
Maria answers only from provided sources when possible.
